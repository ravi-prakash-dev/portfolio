## About

This is the source for my personal portfolio website. It's a single-page, single-file static site — no build step, no dependencies to install, deployed directly to Vercel from this repo.

## Tech

- **HTML / CSS / Vanilla JS** — single-file, no bundler
- **Tailwind CSS** via CDN
- **Google Fonts** — Inter + JetBrains Mono
- **Custom animations** — CSS keyframes + IntersectionObserver for scroll reveals
- **Hosted on** Vercel
